# /images — MS PCB Design

Place optimized images in this folder.

## Recommended Images

| Filename            | Usage                         | Size       |
|---------------------|-------------------------------|------------|
| logo.png            | Brand logo (transparent bg)   | 400×130px  |
| logo-white.png      | Footer / dark bg variant      | 400×130px  |
| favicon.ico         | Browser tab icon              | 32×32px    |
| og-image.jpg        | Open Graph / social preview   | 1200×630px |
| hero-pcb.jpg        | Hero section background       | 1920×1080px|
| about-team.jpg      | About page team photo         | 900×600px  |
| pcb-process.jpg     | Process page illustration     | 900×600px  |

## Optimization Notes
- Use WebP format for photos where possible
- Compress PNG files with TinyPNG or similar
- Always add descriptive alt text in HTML
- Use width/height attributes to prevent layout shift (CLS)
